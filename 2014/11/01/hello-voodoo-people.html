Hello voodoo people

Yeah, i do not know much about voodoo. So in this blog i mostly concentrate on things i do know - things like how to make some cool stuff on the web, which tools to use and of course Ubuntu Linux which i love! As i am not a native english speaker, i hope you forgive me all the spelling and styling mistakes i will make.

The tech department of this blog

When i started to think about creating a personal blog, the first thing i looked for was a simple theme and easy to use blog engine. I have made several webpages using PyroCMS and also have some experience with Wordpress, but none of theme had a suitable theme for personal blogging. So i had to write something that suits my needs - Publisher theme.

To write the Publisher HTML5 theme, i used tools like Sass, Gulp, Compass with its extension and other open source projects. As i could not find a simple blog engine, i decided to use Jekyll static page generator.

In time i will blog about all these tools as on many occasions the official documentation sucks big time! Like with Jekyll i had to google a lot, to get the job done!

The writing department

For coding i use Sublime text 3 editor with several extensions and personal customization. For posting i use UberWriter markdown editor. And yes, my OS of choice is Ubuntu Linux 14.04 LTS. Believe it or not, but it is better than a Mac. Windows does not qualify as an operating system.

What to expect?

Expect to see lot of posts on all subjects mentioned above. Also lot of Laravel and general PHP stuff. It is extremely unlikely you will see here anything about Britney Spears or Christina Aguilera.
